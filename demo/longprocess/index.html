Circus Modal View

Explanations

There are 2 possibilities in this tour.

First case: The modal does NOT show, in this case, the framework will check if there is a Proceed condition: '!condition:params'. If it exists, the framework will see if 'skip' attribute is set? If 'skip' attribute exists, the framework will skip to the step number as indicated, otherwise it will skip to the next step.

Second case: When the modal show up, the framework will proceed normally. However, due to the 'skip' attribute is set, the Skip button will also show up in the tour bubble. To prevent this, set 'noSkip' attribute to 'true'. The Skip button in the bubble can be quite useful for branching steps.
