Explanations

Instead of writing out tour object for each language, in the 'content' attribute of each step, you just need to provide a JSON key. This key should correspond to a key-value pair in translationList object which is provided to the framework when running the tour. The framework will substitute the translated string into the tour object.
This method allows you to maintain multiple translated files for ONLY 1 tour description.
