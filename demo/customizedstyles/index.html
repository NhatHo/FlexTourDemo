Explanations

Recommendations: In order to create your own styles, a short and easy way would be to "@import 'flextour.less'" file and override the variables with your own .less file. However, when compiling .less to .css file please add:
'lessc --relative-urls'.

To Create customized buttons in tour bubble, you have to create an array of objects. Each object is a button and they will be added from 'left -> right' in the bubble. Each object contains 3 attributes:

  1. 'name': String -> The name of the button.
  2. 'style': String -> Contains the classes that will be added to style the button.
  3. 'onclick': Function|String -> This can be either a function that will be triggered when users click on the button. There are 4 default strings that you can use: "Next", "Done", "Skip" and "Back" (Case-sensitive). When these default strings are used, the framework will attach directional function in the customized button.
